# Candy Factory Sales Dashboard | Power BI

This project is an interactive **Candy Factory Sales Dashboard** developed using **Microsoft Power BI** to analyze sales performance and business insights. The dashboard provides a clear and visually engaging overview of key metrics, helping users understand sales trends and make data-driven decisions.

The dashboard includes important KPIs such as **Total Sales, Total Profit, Total Orders, and Quantity Sold**. It also features interactive visualizations to analyze sales by **product category, candy type, region, customer segment, and time period**. Users can explore monthly and yearly sales trends, identify top-performing products, compare regional performance, and monitor overall business growth.

Power BI features such as **DAX measures, calculated columns, slicers, filters, and interactive charts** were used to create dynamic reports that allow users to explore the data efficiently. The dashboard focuses on transforming raw business data into meaningful insights through effective data visualization.

This project demonstrates my skills in **data cleaning, data modeling, DAX, data visualization, and dashboard design** using Power BI. It highlights my ability to convert complex datasets into interactive business reports that support better decision-making.

**Skills Used:** Power BI, Power Query, DAX, Data Modeling, Data Cleaning, Data Visualization, Interactive Dashboard Design, Business Intelligence.
